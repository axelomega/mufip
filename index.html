MuFiP

Multiprecision Fixed Point Number Library for C++11 (MuFiP)

Download as .zip Download as .tar.gz View on GitHub

Motivation

I have not so far found a good library for fixed point arithmetics in C++ that is usefull for bitmunipulation and also integrate well with normal C++ types.

Intended Structure

This library should support fixed size signed and unsigned integeras as well as signed and unsgiend fractional numbers. It might even include a signed unlimited bitfield type. All types should be interoperable with eachother in arithmetich expressions with deterministic results and type resolutions.

Bit Operations

An important aspect of the library is to be able to support bit and bit field operations as is done in e.g. Verilog or VHDL.
